template<typename ADR_TYPE, typename LINK_TYPE>
struct WorkArena< ADR_TYPE, LINK_TYPE >

The WorkArena template struct accepts an address type and a WorkLink struct type and represents a buffer of work links of the given class indexable by the given address type.
